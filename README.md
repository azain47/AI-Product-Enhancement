## 🚀 Installation

To run the Streamlit Web-UI on your own computer, follow these steps:

1. Clone the repository:

    ```sh
    git clone https://github.com/azain47/AI-Product-Enhancement.git
    ```

2. Create a new conda environment:

    ```sh
    conda create -n env
    ```

3. Activate the conda environment:

    ```sh
    conda activate env
    ```

4. Install the required dependencies:

    ```sh
    pip install -r requirements.txt
    ```

5. Run the Streamlit app:

    ```sh
    streamlit run app.py
    ```

## 🎨 Usage

The software provides easy to understand UI which naturally guides on how it works. Upload the desired image, select the object, and generate the enhanced image with desired parameters.
